# 수정공모가 산정 방법

## 1. 수정주가란?

주식시장에서 **종가(종가)**는 해당일 거래 마감 시점의 실제 체결 가격입니다. 그런데 기업이 **액면분할, 무상증자, 유상증자, 배당** 등의 기업 이벤트(Corporate Action)를 실시하면, 주가에 인위적인 단절이 발생합니다.

예를 들어:
- **2:1 액면분할** → 주가가 절반으로 떨어지지만, 실제 기업 가치는 변하지 않음
- **무상증자(100%)** → 주가가 절반으로 떨어지지만, 주주 보유 주식 수는 2배로 증가

이런 단절을 보정하여 **시간에 걸쳐 일관된 비교가 가능하도록** 조정한 가격이 **수정주가(Adjusted Price)**입니다.

## 2. 수정계수란?

**수정계수(Adjustment Factor)**는 기업 이벤트에 의한 주가 변동을 보정하기 위한 계수입니다.

```
수정주가 = 종가 / 수정계수
```

| 이벤트 | 수정계수 예시 | 의미 |
|--------|-------------|------|
| 이벤트 없음 | 1.0 | 종가 = 수정주가 |
| 2:1 액면분할 | 2.0 | 분할 전 종가를 2로 나눠 현재와 비교 가능하게 함 |
| 5:1 액면분할 | 5.0 | 분할 전 종가를 5로 나눠 현재와 비교 가능하게 함 |
| 무상감자(2:1) | 0.5 | 감자 전 종가를 0.5로 나눠(2배) 현재와 비교 가능하게 함 |

수정계수가 1보다 크면 → 주식 수가 늘어난 이벤트 (분할, 무상증자 등)
수정계수가 1보다 작으면 → 주식 수가 줄어든 이벤트 (무상감자, 액면병합 등)

## 4. 수정공모가 산정 공식

```
수정공모가 = 공모가 ÷ 수정계수
```

DataGuide의 수정주가는 과거 종가를 현재 기준으로 보정한 것이므로, 공모가 역시 동일한 수정계수로 나누어야 수정주가와 일관된 비교가 가능합니다.

### 산정 예시

| 종목 | 공모가(원) | 수정계수 | 수정공모가(원) | 이벤트 |
|------|-----------|----------|--------------|--------|
| 펨트론 | 8,000 | 2.0 | 4,000.00 | 액면분할 추정 |
| HPSP | 25,000 | 4.0 | 6,250.00 | 액면분할 추정 |
| 모아데이타 | 20,000 | 5.998 | 3,334.51 | 액면분할 추정 |
| 루닛 | 30,000 | 2.071 | 14,484.01 | 복합 이벤트 |
| 한주라이트메탈 | 3,100 | 1.408 | 2,201.72 | 복합 이벤트 |

수정계수 = 1.0인 종목(255개)은 기업 이벤트가 없으므로 공모가 = 수정공모가입니다.

## 5. 왜 수정공모가를 사용하는가?

본 분석에서 수익률은 다음과 같이 계산됩니다:

```
수익률(%) = (수정주가 ÷ 수정공모가 - 1) × 100
```

만약 수정주가와 원래 공모가를 비교하면:
- 2:1 액면분할한 종목의 수정주가는 분할 후 기준이므로, 원래 공모가 대비 **수익률이 절반으로 과소평가**됨
- 무상감자한 종목은 반대로 **수익률이 과대평가**됨

수정주가 기반의 분석에서는 반드시 공모가도 동일한 수정계수로 보정해야 정확한 수익률이 산출됩니다.

## 6. 적용 범위

| 분석 항목 | 변경 전 | 변경 후 |
|-----------|--------|--------|
| IPO 수익률 (상장일/6M/12M/18M/24M) | 공모가 대비 | 수정공모가 대비 |
| IPO 시가총액 (ipo_mcap) | 공모가 × 최초상장주식수 | 수정공모가 × 최초상장주식수 |
| 공모금액 (ipo_offering) | 공모가 × 공모주식수 | 수정공모가 × 공모주식수 |

## 7. 데이터 현황

- 전체 분석 대상: **405개** 코스닥 신규상장 종목
- 수정계수 ≠ 1인 종목: **150개** (기업 이벤트 발생)
- 수정계수 = 1인 종목: **255개** (이벤트 없음, 공모가 = 수정공모가)
- 수정계수 매칭 실패: **0건**
